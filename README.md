# SportsClubs
## Конечные точки
### Добавить новый клуб ＋
Добавить новый клуб используя следующий запрос:
```
POST /clubs
{
    "name": "Динамо",
    "quantity_tituls": 9,
    "average_age_players": 26.3,
    "town_id": 1
}
```

### Изменить данные о клубе ✎
Изменить данные о клубе используя слудеющий запрос:
```
PUT /clubs/id
{
    "name": "ЦСКА",
    "quantity_tituls": 11,
    "average_age_players": 25.5,
    "town_id": 3
}
```

### Удалить клуб ❌
Удалить клуб используя слудеющий запрос:
```
DELETE /clubs/id
```

### Получить весь список клубов 👁
Получить весь список клубов используя слудеющий запрос:
```
GET /clubs
```

### Получить данные о клубе по id 📄
Получить данные о клубе по id используя слудеющий запрос:
```
GET /clubs/id
```

### Поиск данных по клубу 🔍️
Поиск данных по клубу используя слудеющий запрос:
```
GET /club/search?search=name_club
```
